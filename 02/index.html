Визначення тегів

Тег strong Призначений для відення напису

Тег em Призначений для надання акценту тексту за допомогою курсиву

Тег small Призначений для зменьшення шрифта написа

Тег ins Призначений для підкреслення тексту

Тег del Призначений для закреслювання тексту

Тег sub Призначений для відображення тексту у вигляді нижнього індексу

Тег sup Призначений для відображення тексту у вигляді верхнього індексу

Тег code Призначений для відображення одної чи декількох строк програмного коду.

Тег kbd Призначений для позначення тексту,що воодиться з клавіатур або для назви клавіш

Тег samp Використовується для відображення тексту,який являється результатом вивода комп'ютерної програми чи скрипта

Тег pre 

 Визначає блок попередньо відформатованого тексту

Тег dfn Як правило коли в документі з'являється новий термін,він виділяється курсивом,і дається його визначення

Тег abbr АБР використовується коли потрібно виділити абревіатуру

Тег q Використовується для виділення цитат. вміст в браузері відображається в лапках

Тег cite Помічає текст як цитату чи зноску на інший матеріал.

Тег blockquote

Призначений для виділення дових цитат в середині документа.Текст,обозначений цим тегом,традиційно відображається як вирівняний блок з відступами зліва та справа,а також відбивкою зверху та знизу

Тег br
Встановлює перенесення строки в тому місці,де знаходиться,різниця з тегом p в тому,що використання тега br не додає відступ перед строкою

Тег hr - малює горизонтальу лінію,яка за своїм видом залежить від використовуємих параметрів,а також браузера

Тег ul - встановлює маркований список

Тег ol - встановлює нумерований список

Тег li - визначає окремий елемент списку

a href=“” - задає адресу документа, на який слід перейти.

img - призначений для відображення на веб-сторінці зображень в графічному форматі GIF, JPEG або PNG.
