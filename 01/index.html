Структура найпростішого html-документу

<!DOCTYPE html> - специфікація версії HTML

<html> - Контейнер вмісту Веб-сторінки

<head> - Голова документу, призначена для браузера та пошукових систем

<title> - Сайт про HTML - назва документу, що відображається вгорі вікна браузера

</head> - Кінець голови документу

<body> - Тіло документу, що зберігає вміст веб-сторінки

</body> - Кінець тіла документу

</html> - Кінець вмісту Веб-сторінки
